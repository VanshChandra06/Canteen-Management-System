# Canteen-Management-System

A simple command-line Canteen Management System built with Python and MySQL.
The project automatically creates the canteen database, sets up all required tables, inserts sample data, and provides a menu-driven interface to run predefined queries.
Uses mysql-connector library in Python.
MySQL is run via XAMPP.

How to Run:

1. Install Python and XAMPP on your device
2. Open XAMPP Control Panel
3. Run Apache and MySQL
4. Open terminal and install requirements:-
   pip install -r requirements.txt
5. Run main.py
